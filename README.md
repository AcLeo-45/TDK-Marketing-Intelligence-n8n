## 📌 Overview

This project is a **Marketing Intelligence Automation Workflow** built using **n8n**. It automatically ingests marketing data from a MySQL database, calculates key business metrics, and delivers a formatted HTML report via email — every day at 8:00 AM.

---

## 🎯 Objective

Build an end-to-end automated workflow that:
- Pulls campaign, leads, spend, and conversion data from MySQL
- Calculates CAC, ROI, Conversion Rate, and Lead Quality Score
- Sends a daily automated email report with all metrics

---

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| **n8n** (Cloud) | Workflow orchestration |
| **MySQL** (Railway) | Cloud database |
| **Gmail** | Automated email reporting |
| **JavaScript** | Metric calculations inside n8n Code node |

---

## 📊 Database Tables

| Table | Description |
|-------|-------------|
| `users` | Company accounts and regions |
| `campaigns` | Campaign details, budget, status |
| `leads` | Lead info, quality score, source |
| `conversion_events` | Clicks, purchases, demo requests |
| `marketing_spend` | Daily spend, impressions, clicks |

> Full data definitions available in `Data_Definitions.csv`

---

## ⚙️ Workflow Structure

```
Schedule Trigger (Daily 8 AM)
        ↓
MySQL Node 1 — Fetch Campaigns + Spend Data
        ↓
MySQL Node 2 — Fetch Leads + Revenue Data
        ↓
Code Node (JavaScript) — Calculate Metrics
        ↓
Gmail Node — Send HTML Email Report
```

---

## 📐 Metrics Calculated

| Metric | Formula |
|--------|---------|
| **CAC** (Customer Acquisition Cost) | Total Spend ÷ Total Conversions |
| **ROI** (Return on Investment) | (Revenue - Spend) ÷ Spend × 100 |
| **Conversion Rate** | Conversions ÷ Clicks × 100 |
| **Avg Lead Quality Score** | Average of lead_quality_score per campaign |

## 🚀 How to Run

### 1. Setup Database
- Create a MySQL database (Railway.app recommended)
- Run `raw.sql` to create tables and insert sample data

### 2. Import Workflow
- Open your n8n instance
- Go to **Workflows** → **Import**
- Upload `TDK_Marketing_Intelligence_Workflow.json`

### 3. Configure Credentials
- Add your **MySQL credentials** (host, port, user, password, database)
- Connect your **Gmail account** via OAuth2
- Update the **"To" email** in the Gmail node

### 4. Activate Workflow
- Click **Publish** in n8n cloud
- Workflow will run automatically every day at 8:00 AM

---

## 💡 Assumptions

- Sample data from `raw.sql` is used for demonstration purposes
- In a production environment, live data would be inserted into MySQL daily, resulting in updated reports each run
- Email is configured to send to a designated stakeholder. Update the Gmail node "To" field as needed
- ROI shows -100% for campaigns with no purchase conversion events in the sample data — this is expected behavior with the provided dataset
